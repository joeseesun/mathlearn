6平面直角坐标系

P(x, y)关于 x 轴P₁(x, −y)
关于 y 轴P₂(−x, y)
关于原点P₃(−x, −y)
对称点探索：点击（或拖动）坐标系任意位置放置点 P
虚线表示对称关系：P 与 P₁ 的连线被 x 轴垂直平分，P 与 P₂ 的连线被 y 轴垂直平分，P 与 P₃ 的连线经过原点。当 P 落在坐标轴上时，部分对称点会重合。

小提示：关于 x 轴对称的点，横坐标相同、纵坐标互为相反数；关于 y 轴对称正好反过来；关于原点对称时，横、纵坐标都互为相反数。
